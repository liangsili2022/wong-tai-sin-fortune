# 黄大仙灵签

一个基于网页的黄大仙灵签抽签应用，支持随机抽签、查看签文和解签，以及浏览全部100支灵签。

## 功能特点

- 🧧 随机抽签，模拟真实抽签仪式
- 📜 显示签文、吉凶等级和解签内容
- 🔄 支持"再求一签"功能
- 📖 可查看全部100支灵签
- 📱 响应式设计，适配手机和桌面端

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- JSON 数据存储

## 使用方法

1. 克隆或下载项目到本地
2. 用浏览器打开 `index.html` 文件
3. 点击"求签"按钮进行抽签
4. 点击"再求一签"重新抽签
5. 点击"查看全部签文"浏览所有灵签

## 部署到 GitHub Pages

1. 在 GitHub 上创建一个新仓库
2. 将所有项目文件推送到仓库
3. 进入仓库设置，找到 "Pages" 选项
4. 选择 "main" 分支和 "/root" 目录
5. 点击保存，等待几分钟后即可通过 GitHub Pages 访问应用

